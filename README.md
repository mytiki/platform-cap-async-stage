# Aggregation: Async Stage
[AWS S3](https://aws.amazon.com/s3/) based staging environment for consolidating asynchronous writes from [async-endpoint](https://github.com/mytiki/platform-cap-async-endpoint) before loading into mytiki.com data lake compatible tables with [async-write](https://github.com/mytiki/platform-cap-async-write).
